# Writer Education

A single-page reference table of 100 famous writers of the last 200 years: novelists, poets, playwrights, and journalists. Each entry lists birth/death dates, highest formal education, day job, and major works, sourced from Wikipedia.

Rows with no completed university degree (or who were self-taught) are highlighted.

Live page: published via GitHub Pages.

Open `index.html` locally, or visit the Pages URL.
